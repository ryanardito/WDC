HTML

Hypertext Markup Language

Pengertian

Bahasa yang digunakan untuk membuat sebuah halaman web, yang kemudian dapat diakses untuk menampilkan berbagai informasi di dalam sebuah penjelajah web Internet (Browser).

Manfaat

1. Dapat Membuat Halaman Website
2. Dapat mengaplikasikan pikiran dalam bentuk website
3. Dapat Menghasilkan Banyak Uang
